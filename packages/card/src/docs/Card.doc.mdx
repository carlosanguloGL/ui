# Cards

Display a card with different atributes, variants and sizes.

[Github repo](https://github.com/unicoderns/ui/tree/master/packages/card)

```html
<ui-card
  variant="primary"
  header="header"
  title="title"
  background="true"
  style="width: 18rem;"
  img-src="https://picsum.photos/600/300/?image=25"
  img-bottom="false"
  footer="footer"
></ui-card>
```

### Card text

This component is use to send text to the card using its corresponding CSS class

```html
<ui-card-text>
  Some quick example text to build on the card title and make up the bulk of the
  card's content.
</ui-card-text>
```

## Props

| Name         | Type      | Description                                       |
| ------------ | --------- | ------------------------------------------------- |
| `variant`    | `string`  | variant class defined in the theme                |
| `outline`    | `boolean` | will use outline variants if true                 |
| `style`      | `string`  | allows to change card's size                      |
| `title`      | `string`  | allows to add a title to the card                 |
| `subtitle`   | `string`  | allows to add subtitle to the card                |
| `header`     | `string`  | allows to add header to the card                  |
| `footer`     | `string`  | allows to add footer to the card                  |
| `imgSrc`     | `string`  | allows to add a image to the card                 |
| `imgAlt`     | `string`  | allows to add an alternative text to card's image |
| `imgBottom`  | `boolean` | card will show the image at the bottom if true    |
| `centerText` | `boolean` | center all text inside the card                   |

## Slots

| Name      | Description       |
| --------- | ----------------- |
| `default` | card text content |

# Theme

Use the next `json` to define a new theme for the component

```json
{
    cssClass: {
    main: string // main class
    header: string // css class for header of the card
    body: string // css class for the body of the card
    title: string // css class for title of the card
    subtitle: string // css class for subtitle of the card
    img: string // css class for image of the card
    imgBottom: string // css class for image of the card when on bottom
    footer: string // css class for footer of the card
    center: string // css class to center text
    text: string // css class for main text of the card
    variants: { // define a custom list of variants
      'variant1': string,
      'variant2': string,
      ...
      'variantN': string,
    },
    textVariants: { // define a custom list of text variants
      'variant1': string,
      'variant2': string,
      ...
      'variantN': string,
    },
    outlineVariants: { // must match the same variants as before
      'variant1': string,
      'variant2': string,
      ...
      'variantN': string,
    },
    outlineTextVariants: { // must match the same text variants as before
      'variant1': string,
      'variant2': string,
      ...
      'variantN': string,
    },
  }
}
```

Check the page [what is a theme](/docs/docs-theming--page#what-is-a-theme) to see how to use this json
