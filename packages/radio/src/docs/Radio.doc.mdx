# Radios
Display an radio with text to users.

[Github repo](https://github.com/unicoderns/ui/tree/master/packages/Radio)

```html
    <ui-radio :="args" :show="toggle" @show="show" @close="close" @hide="hide">{{ args.default }}</ui-radio>
```

## Props

| Name | Type  | Description |
| ---- | ---- | ----------- |
| `variant` | `string` | variant class defined in the theme |

## Slots

| Name | Description |
| ---- | ----------- |
| `default` | actual span content |

# Events
Radio will emit this events while changing state

| Event | Params | Desription |
| ---- | ---- | ----------- |
| Toggle | active: `boolean` | Will only raise when `toggle = true` |

# Accessibility
Change the next properties to change the default accessibility text

| Name | Property | Default |
| ---- | ---- | ----------- |
| role | aria:role | `radio` |

Check the page [what is aria](/docs/docs-aria--page#what-is-aria-config) to see how to config accessibility globally for radios.

# Theme
Use the next `json` to define a new theme for the component

```json
{
  cssClass: {
    main: string, // main class
    variants: { // define a custom list of variants
      'variant1': string,
      'variant2': string,
      ...
      'variantN': string,
    },
  },
}
```

Check the page [what is a theme](/docs/docs-theming--page#what-is-a-theme) to see how to use this json
