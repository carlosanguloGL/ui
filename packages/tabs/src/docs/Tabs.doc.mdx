# Tabs
Toggle between tabs and display content according the selected tab. It includes support for active and disabled states tabs.

[Github repo](https://github.com/unicoderns/ui/tree/master/packages/tabs)

## Props

### Select

| Name | Description |
| ---- | ---- |
| `select` | Callback when the index of selected tab changes. |

```html
<ui-tabs @select="select">
  <ui-tab title="Tab1" :active="true" :disabled="false">
    Tab 1 content
  </ui-tab>
</ui-tabs>
```

### Active
| Name | Description | Type |
| ---- | ---- | ---- |
| `active` | Indicates that the tab is selected and shows its content. | `Boolean` |


### Disabled
| Name | Description | Type |
| ---- | ---- | ---- |
| `disabled` | When a tab is disable it can't be selected. | `Boolean`

```html
<ui-tabs @select="select">
  <ui-tab title="Tab1" :active="true" :disabled="false">
    Tab 1 content
  </ui-tab>
  <ui-tab title="Tab2" :active="false" :disabled="true">
    Tab 2 content
  </ui-tab>
</ui-tabs>
```

## Each Tab Props
| Name | Type  | Description |
| ---- | ---- | ----------- |
| `title` | `string` | tab title |
| `active` | `boolean` | if the tab is selected |
| `disabled` | `boolean` | if the tab is disabled |


## Events
Tab panel will emit this events while changing state

| Event | Params | Desription |
| ---- | ---- | ----------- |
| select | `index: number` | Will raise when a tab is selected |


## Each Tab Slots
| Name | Description |
| ---- | ----------- |
| `default` | actual tab content |

## Accesibility
| Name | Property | Value | Description |
| ---- | -------- | ----- | ----------- |
| role | aria:role | `tab` | It indicates that it is a tab |
| currentPage | aria:current | `page` |It indicates which tab is active |

```html
<ul class="nav nav-tabs">
  <li class="nav-item">
    <a class="nav-link active" aria-current="page" role="tab">
      Tab 1
    </a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" role="tab">
      Tab 2
    </a>
  </li>
</ul>
```

## Theme
Use the next `json` to define a new theme for the component

```json
{
  cssClass: {
    main: string // main tab panel (ul) class
    item: string // each tab class (li)
    link: string // each tab link class (a)
    itemActive: string // class for active tab
    itemDisabled: string // class for disabled tab
  }
}
```

Check the page [what is a theme](/docs/docs-theming--page#what-is-a-theme) to see how to use this json
