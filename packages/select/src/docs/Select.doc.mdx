# Select

Custom select that admits one or multiple selections with different sizes, and disabled option.

[Github repo](https://github.com/unicoderns/ui/tree/master/packages/select)

```html
<ui-select
  :datasource="datasource"
  multiple
  size="md"
  disabled
  v-model="selectedIndex"
></ui-select>
```

# Props

| Name         | Type                         | Description                                  |
| ------------ | ---------------------------- | -------------------------------------------- |
| `datasource` | `array`                      | array with select options [(number, string)] |
| `multiple`   | `boolean`                    | allow to select multiple options if true     |
| `size`       | `PropType<InputSizeVariant>` | change select size                           |
| `disabled`   | `boolean`                    | disable select if true                       |

# Binding

Select supports two way data binding, just use `v-model="variable"` as prop to pass the reference that you want to bind.

The ref `variable` could be a number if it's a simple selection or could be an array if your using multiple selection.

# Accessibility

Change the next properties to change the default accessibility text

| Name  | Property   | Default  |
| ----- | ---------- | -------- |
| role  | aria:role  | `select` |
| label | aria:label | `null`   |

Check the page [what is aria](/docs/docs-aria--page#what-is-aria-config) to see how to config accessibility globally for buttons.

# Theme

Use the next `json` to define a new theme for the component

```json
{
  cssClass: {
    main: string  // main class
    sizes: { /// class for each size variant: md, sm, lg.
      [InputSizeVariants.Large]: string
      [InputSizeVariants.Medium]: string
      [InputSizeVariants.Small]: string
    }
  },
}
```

Check the page [what is a theme](/docs/docs-theming--page#what-is-a-theme) to see how to use this json
