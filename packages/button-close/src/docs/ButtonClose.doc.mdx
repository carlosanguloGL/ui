# Button Close

[Github repo](https://github.com/unicoderns/ui/tree/master/packages/button-close)

```html
<ui-button-close :disabled="false" :invert="false" @close="close" />
```

# Sample
<Canvas>
  <Story id="ui-closebutton--default"/>
</Canvas>


## Props

| Name | Description | Type |
| ---- | ----------- | ---- |
| `disabled` | If true, the button won't be clickable. | `boolean` |
| `invert` | It adds the `inverted` class variant. | `boolean` |

## Events
Close button will emit a callback when is clicked.
| Name | Description | Type |
| ---- | ----------- | ---- |
| `close` | Callback when the button is clicked. | `callback` |

## Accessibility
| Name | Property | Default | Description |
| ---- | -------- | ----- | ----------- |
| label | aria:label | `close` | Indicates that it is a closable element |

## Theme
Use the next `json` to define a new theme for the component

```json
{
  cssClass: {
    main: string // main button class
    inverted: string // class for inverted button
  }
}
```

Check the page [what is a theme](/docs/docs-theming--page#what-is-a-theme) to see how to use this json
