# Inputs

Custom input with label, includes different types and disabled option.

[Github repo](https://github.com/unicoderns/ui/tree/master/packages/input)

```html
<ui-input label="Email address" type="email" disabled="false"></ui-input>
```

## Types

Supports email, password and text, also it can be disabled.

## Props

| Name       | Type      | Description                                   |
| ---------- | --------- | --------------------------------------------- |
| `type`     | `string`  | define input's type: email, password or text. |
| `label`    | `string`  | adds a label to the input                     |
| `disabled` | `boolean` | disabled input if true                        |

# Binding

Select supports two way data binding, just use `v-model="variable"` as prop to pass the reference that you want to bind.

# Accessibility

Change the next properties to change the default accessibility text

| Name        | Property         | Default |
| ----------- | ---------------- | ------- |
| describedby | aria:describedby | null    |

Check the page [what is aria](/docs/docs-aria--page#what-is-aria-config) to see how to config accessibility globally for inputs.

# Theme

Use the next `json` to define a new theme for the component

```json
{
  cssClass: {
    main: string // main class
    label: string // label css class
  },
}
```

Check the page [what is a theme](/docs/docs-theming--page#what-is-a-theme) to see how to use this json
